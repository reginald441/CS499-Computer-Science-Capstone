Selected Work

Capstone Artifacts

01

Software Design and Engineering

Android Inventory Tracker

Originally developed in CS 360: Mobile Architecture and Programming

This Android application supports user registration, login, inventory creation, item management, and SQLite database interaction. The project demonstrates mobile application development, interface design, data persistence, validation, and software organization.

Current Enhancement Plan

  • Refactor the application toward an MVVM architecture.
  • Separate the user interface, business logic, and database operations.
  • Improve input validation, error handling, accessibility, and maintainability.
  • Strengthen comments, testing, and technical documentation.

Current status: Enhancement in progress

02

Algorithms and Data Structures

Corner Grocer

Originally developed in CS 210: Programming Languages

This C++ application reads grocery purchase records, counts item frequencies, supports searching, and produces reports. The project demonstrates data processing, searching, reporting, file handling, and the practical use of data structures.

Current Enhancement Plan

  • Implement a self-balancing red-black tree for organized data storage.
  • Create at least two detailed report formats.
  • Compare the enhanced structure with the original map-based approach.
  • Evaluate runtime, space complexity, scaling, and memory trade-offs.

Current status: Enhancement planned

03

Databases

Grazioso Salvare Dashboard

Originally developed in CS 340: Client/Server Development

This dashboard connects a user-facing application to MongoDB and supports CRUD operations, filtering, and data visualization. The project demonstrates database integration, query design, client/server communication, and dashboard development.

Current Enhancement Plan

  • Create indexes for frequently queried MongoDB fields.
  • Improve query performance, filtering, and error handling.
  • Validate and sanitize user-supplied values.
  • Reduce NoSQL injection risk and strengthen secure database access.

Current status: Enhancement planned
